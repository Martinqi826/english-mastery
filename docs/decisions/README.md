# 架构决策记录 (ADR)

本目录存放详细的架构决策记录。

主要决策请参考 [ARCHITECTURE.md](../../ARCHITECTURE.md)

## 如何添加新的决策记录

1. 创建新文件：`YYYYMMDD-决策标题.md`
2. 使用以下模板：

```markdown
# ADR-XXX: 决策标题

## 状态
提议 / 采纳 / 废弃 / 替代

## 背景
为什么需要做这个决策？

## 决策
具体选择了什么？

## 后果
这个决策带来的影响是什么？
```
